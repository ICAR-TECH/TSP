# TSP: Time Series Projection Software

## Website link: https://icar-tech.github.io/TSP/

**TSP (Time Series Projection)** is a user-friendly tool for analyzing time series data and projecting future trends based on historical values. Developed collaboratively by **ICAR-NIAP, New Delhi** and **ICAR-IASRI, New Delhi**, the software simplifies statistical modeling and forecasting for researchers, analysts, and decision-makers.

## Key Features

* **Simple Data Input:** Paste time stamps and corresponding values directly.
* **Multiple Models:** Linear, Second-Order Polynomial, and Exponential regression.
* **Projection Control:** Define projection range and adjust historical data usage.
* **Graphical Output:** Visual comparison of original and projected values.
* **Export Options:** Save projections as images and CSV files.

## Applications

* Agricultural planning and yield forecasting
* Economic and market trend analysis
* Environmental and climate studies
* Scientific research and experimentation

## Benefits

* Easy-to-use interface
* Flexible model selection
* Reliable projections
* Shareable outputs

Planned enhancements include integration with machine learning models, real-time data support, and improved visualization features.
